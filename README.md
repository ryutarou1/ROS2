# ROS2 Listener and Talker

## 概要
ROS2を使用したlistenerとTalkerのプロジェクトです。Listenerは"countup"というトピックからInt16メッセージを購読し、そのデータをログに表示します。Talkerは"countup"トピックにInt16メッセージを定期的にパブリッシュし、データを1ずつ増やします。
